# Milestone 4 Pro — Sprint 4-3 點名管理系統

## 完成功能
- 新增 `#/attendance-admin` 領隊點名管理頁。
- 建立、編輯與刪除集合／登車／用餐／回飯店點名場次。
- 團員可標記已到、遲到、未到與未確認。
- 支援姓名、房間、去回程機位搜尋。
- 支援依名單、房間及機位排序。
- 即時計算已到、遲到、未到與未確認人數。
- 支援全部已到、全部重設及只看未到。
- 點名場次與紀錄統一儲存在 `amt-m4-pro-data-v1`。
- 保留舊版每日簽到資料遷移結果。

## 使用方式
開啟 `#/pro` 後選擇「4-3 點名管理」，或直接開啟 `#/attendance-admin`。
