# GitHub Desktop 更新方式

1. 解壓縮 ZIP。
2. 打開解壓縮後的 `Malaysia-Travel-Companion-v2`。
3. 複製資料夾內的全部內容。
4. 貼到本機 Repository：
   `桌面/GitProjects/Malaysia-Travel-Companion-v2/`
5. 出現同名檔案時，選擇全部取代。
6. 打開 GitHub Desktop。
7. Summary 輸入：
   `Milestone 1 Complete`
8. 按 `Commit to main`。
9. 按 `Push origin`。
10. 到 GitHub → Settings → Pages，設定 main / root。
