# Malaysia Travel Companion v6.1 — Phase 2

## Unified Interface

本階段在 Phase 1 視覺重構基礎上，將所有既有頁面統一為同一套易讀、易點擊、高對比介面。

### 更新內容
- 全站文字、標題、說明與表單尺寸統一。
- 所有按鈕與輸入元件採 54–56px 以上觸控高度。
- 導覽、地圖、通知、數位手冊、QR、領隊管理等舊模組全面套用一致字級。
- 強化焦點框、色彩對比、卡片圓角與留白。
- 手機窄螢幕改為單欄優先，避免內容過度擁擠。
- 不再區分一般模式與長輩模式。
- PWA 快取更新為 Phase 2。
