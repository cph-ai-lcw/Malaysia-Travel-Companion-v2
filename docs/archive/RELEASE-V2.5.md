## GitHub Desktop 更新方式
1. 解壓縮本專案。
2. 在 GitHub Desktop 點 Repository → Show in Explorer。
3. 將本資料夾內全部檔案複製到 Repository 根目錄並覆蓋。
4. 回到 GitHub Desktop，確認 Changes。
5. Summary 填寫：Release v2.5 – Complete Travel Companion
6. Commit to main → Push origin。
