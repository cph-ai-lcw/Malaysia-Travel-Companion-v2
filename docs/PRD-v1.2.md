# GN Malaysia Travel Handbook 2026 — PRD v1.2

## Scope
This release improves navigation while preserving the two-language system introduced in v1.1.

## Navigation
- Dedicated Home page and route: `#/home`
- Persistent top-left Home button
- Home tab in fixed bottom navigation
- Direct links such as `#/plan` remain supported
- Unknown or empty routes fall back to Home

## Languages
- Traditional Chinese
- Vietnamese
- No bilingual parallel mode

## Data status
Formal itinerary, LINE group URL, room allocation and flight-seat allocation remain pending import from approved source files.
