# Malaysia Travel Companion v6.1 — Phase 1

## Universal iOS Design Edition

本階段僅重構視覺與操作尺寸，保留原有資料、路由與功能。

- 全站統一大字體、大按鈕、高對比，不再提供獨立大字模式。
- 首頁 Hero 改為吉隆坡雙子星與熱帶度假村主視覺。
- 主要分頁 Icon Tab 浮在 Hero 圖片右下方。
- 卡片統一 Glassmorphism、24px 圓角及柔和陰影。
- 提升導覽、表單與快捷功能的觸控尺寸。
- 支援 iOS Safe Area 與 reduced-motion 無障礙偏好。
- PWA 快取更新為 v6.1 Phase 1。
