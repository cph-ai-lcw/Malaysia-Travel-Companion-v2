# Milestone 5-5：旅行資訊整合＋出發前檢查

完成日期：2026-07-18  
狀態：正式完成  
版本：5.5.0

## 完成內容

- 將旅行資訊集中於 `data/trip/travel-info.js`，支援離線使用。
- 整合航班、住宿、電壓插座、時差、貨幣、語言與緊急聯絡資訊。
- 整合入境海關、行李安全、通訊禮儀、熱帶天氣與團體行動提醒。
- 提供繁體中文／越南文切換、分類瀏覽與搜尋。
- 提供可勾選的出發前檢查表，狀態保存在本機裝置。
- MDAC 改為「公司／旅行社統一協助處理」，一般團員不再顯示自行填寫入口。
- 顯示 MDAC 處理狀態：尚未通知、資料收集中、旅行社處理中、已完成。
- 領隊登入後可更新 MDAC 狀態；一般團員僅可查看。
- 出發前檢查項目改為確認是否收到公司／旅行社 MDAC 通知，避免重複填寫。

## 驗收標準

- `#/travel-info` 可正常開啟並離線瀏覽。
- 五個分類、搜尋、中越文、電話連結及檢查表可正常操作。
- MDAC 頁面明確提醒不要自行重複填寫。
- 領隊模式可更新 MDAC 狀態。
- JavaScript、Service Worker 與 ZIP 完整性檢查通過。
